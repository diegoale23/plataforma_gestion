Descripción del puesto Estamos en búsqueda de un Desarrollador Backend .Net Senior con un mínimo de 5 años de experiencia en el desarrollo de aplicaciones y servicios backend. El candidato ideal será responsable de diseñar, desarrollar y mantener microservicios utilizando tecnologías .NET, asegurando que las aplicaciones sean escalables, seguras y eficientes. Responsabilidades Desarrollar y mantener aplicaciones basadas en microservicios utilizando .NET y Web API REST con ASP.NET Core (versión 3.1 o superior). Implementar ORM (Object-Relational Mapping) utilizando Microsoft Entity Framework. Diseñar e implementar sistemas de cache utilizando Redis u otras soluciones de cache clave-valor para optimizar el rendimiento de las aplicaciones. Trabajar con bases de datos NoSQL como MongoDB y relacionales como Microsoft SQL Server, asegurando una correcta integración y manejo de datos. Contenerizar aplicaciones y APIs utilizando Docker o containerd, garantizando un despliegue eficiente y reproducible. Diseñar y mantener arquitecturas escalables basadas en clústeres de Kubernetes para soportar la carga de trabajo y mejorar la resiliencia de las aplicaciones. Utilizar herramientas de gestión de API como Amazon API Gateway, WSO2 o Apigee, implementando soluciones que faciliten la administración y supervisión de los servicios expuestos. Implementar y gestionar la autenticación de usuarios a través de un servidor de gestión de identidad (IAM) y utilizar estándares como OAuth, JWT y OpenID para asegurar la comunicación entre servicios. Desarrollar y mantener procedimientos ETL utilizando herramientas de integración comerciales u open source, incluyendo SQL Server Integration Services o Pentaho. Especificar y documentar APIs siguiendo la especificación OpenAPI con un enfoque en API First. Implementar herramientas de logging y monitorización para garantizar el correcto funcionamiento y la trazabilidad de las aplicaciones. Experiencia sólida con bases de datos NoSQL (MongoDB) y relacionales (Microsoft SQL Server). Conocimiento en la implementación de microservicios y contenedores (Docker y containerd). Experiencia en plataformas de orquestación de contenedores como Kubernetes. Familiaridad con herramientas de API Management (Amazon API Gateway, WSO2, Apigee). Experiencia en autenticación y autorización de usuarios utilizando estándares Oauth, JWT y OpenID. Conocimiento en herramientas de ETL como SQL Server Integration Services y Pentaho.
